# Octopye Digital Designs

Official static website for Octopye Digital Designs.

## Overview

This repository contains the Octopye marketing site built with plain HTML, CSS, and JavaScript.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

- `index.html` - Homepage
- `services.html` - Services overview
- `packages.html` - Package pricing and add-ons
- `projects.html` - Portfolio/projects
- `estimate.html` - Instant estimator
- `booking.html` - Booking request page
- `styles.css` - Global styles
- `script.js` - Global scripts
- `images/` - Static image assets

## Local Preview

Run a local server from the project root:

```powershell
py -m http.server 5500
```

Then open:

- `http://127.0.0.1:5500/index.html`

## Deployment

This site is static and can be deployed to GitHub Pages or any static hosting provider.
